# mht-unarchive
Python program to extract from `mht` (Microsoft HTML archive) to html and image files.

Inspired by [yangbo/mhtviewer](https://github.com/yangbo/mhtviewer)
Rewritten using `email.message_from_file`

## Usage
```
./extract.py <file.mht>

```

## Testing
```
poetry install
poetry run ptw
```
